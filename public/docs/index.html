Introduction

API do sistema de Almoxarifado

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost

Authenticating requests

This API is not authenticated.

Autenticação

APIs para login e logout do sistema

Cria um novo usuário no sistema

requires authentication

Example request:
const url = new URL(
    "http://localhost/api/cadastrar"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "airizgswyosygaxxqoxtyygcqiuyzupdsjeahvbzfnrqqylubhlhhiwapu",
    "email": "tisvma@prefeitura.sp.gov.br",
    "password": "Teste!123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
import requests
import json

url = 'http://localhost/api/cadastrar'
payload = {

    "password": "Teste!123"
}
headers = {
  'Content-Type': 'application/json',
  'Accept': 'application/json'
}

response = requests.request('POST', url, headers=headers, json=payload)
response.json()

    'http://localhost/api/cadastrar',
    [
        'headers' => [
            'Content-Type' => 'application/json',

        'json' => [
